English for Life

Para instalar la app en cualquier dispositivo,
necesita estar en internet. Sigue estos pasos — es gratis y tarda 3 minutos.

💡 ¿Por qué? Los navegadores solo permiten instalar apps desde sitios web seguros (https://), no desde archivos locales. GitHub Pages te da un sitio gratuito y permanente.
⚡ Método rápido — GitHub Pages (gratis, 3 min)
1Descarga el archivo de la app ya lo tienes
Guarda el archivo english_for_life_pwa.html en tu computador (ya lo descargaste antes).
2Crea cuenta en GitHub gratis
Ve a github.com → clic en Sign up → crea cuenta con tu email.
Si ya tienes cuenta, salta al paso 3.
3Crea un repositorio nuevo
Una vez en GitHub, clic en el botón verde New (o el ícono +).
• Nombre del repositorio:
english-for-life
• Marca ✅ "Add a README file"
• Visibility: Public
• Clic en Create repository
4Sube el archivo HTML
Dentro del repositorio, clic en Add file → Upload files.
Arrastra el archivo english_for_life_pwa.html y renómbralo a:
index.html
Clic en Commit changes (botón verde).
5Activa GitHub Pages gratis
En el repositorio: Settings (arriba) → Pages (menú izquierdo)
→ Source: Deploy from a branch
→ Branch: main → carpeta: / (root)
→ Clic en Save.

Espera 1-2 minutos. Tu app estará en:
https://TU-USUARIO.github.io/english-for-life/
6¡Instala desde esa URL! en cada dispositivo
Abre esa URL en Chrome/Edge (PC o Android) o Safari (iPhone).
Aparecerá el botón de instalar automáticamente. ✅

Comparte el link con toda la familia — cada uno la instala en su dispositivo.
🚀 Alternativa más fácil — Netlify Drop (30 segundos)
1Ve a Netlify Drop
Abre en tu navegador:
https://app.netlify.com/drop
2Arrastra el archivo
Renombra primero el archivo a index.html.
Crea una carpeta nueva → pon el archivo adentro → arrastra esa carpeta a la página de Netlify Drop.
3Obtén tu URL pública
Netlify genera automáticamente una URL tipo:
https://nombre-aleatorio.netlify.app
Abre esa URL en cualquier dispositivo e instala la app.
La app es 100% tuya — el código fuente es un archivo HTML.
Nadie accede a tus datos. GitHub Pages y Netlify son gratuitos.
